Ключи формы (также известные как морфинг) / Blender

Ключи формы (в Verge3D называются морфинг) это свойство, которое позволяет изменять объект в реальном времени. Наиболее распространенным вариантом использования ключей формы является создание параметрических объектов, у которых могут изменяться размеры, форма, толщина, отверстия, контуры и др.

#

Настройка

Ключи формы в Blender экспортируются по умолчанию. В Verge3D поддерживаются только ключи формы типа Relative. Обращаем ваше внимание, что из-за технических ограничений максимальное количество активных ключей формы ограничено. В большинстве случаев, ограничение количества ключей для объектов составит 6 и 5 для объектов со скиннингом (ключ "Basis" не учитывается при подсчёте). Ознакомьтесь с данной темой для получения информации о подсчёте ограничений в более сложных ситуациях.

Во время рендера, движок выбирает 6 ключей формы с самым большим влиянием и игнорирует остальные.

#

Анимация

Чтобы анимировать ключи формы в Verge3D, добавьте ключевые кадры для их параметра Value. Анимация ключей формы работает точно также, как и анимация других параметров. Вы можете воспроизводить анимацию при запуске приложения, а можете использовать для этой задачи animation puzzles.

#

Управление с помощью Puzzles

Чтобы управлять ключами формы в реальном времени существуют два пазла: get morph factor и set morph factor.
