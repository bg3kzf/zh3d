Настройки в Blender

#

Объектная анимация

Ключевые кадры анимации могут быть добавлены в Blender стандартным путём: Object > Animation > Insert Keyframe... menu (или при нажатии горячей клавиши I), после этого, на временной шкале выбирается другой кадр, объект перемещается/вращается/масштабируется и добавляется новый ключевой кадр для этих изменений.

#

Скелетная анимация

Прискиннивание объекта в Blender выполняется с использованием Armature. Verge3D поддерживает максимум 4 веса на одну кость (взаимодействовать с костью будут только 4 веса с самым большим значением влияния, остальные будут проигнорированы). Кости могут быть анимированы в Pose Mode (Ctrl+Tab), используя Pose > Animation > Insert Keyframe... menu (или горячая клавиша I), после чего, на временной шкале выбирается другой кадр, кость перемещается/вращается/масштабируется и с данными изменениями добавляется новый ключевой кадр.

#

Ключи формы или морфинг-анимация

В Blender вы можете добавить ключи формы к объекту стандартным путём (для получения более подробной информации пройдите по ссылке).

Значения ключей формы могут быть анимированы в Blender нажатием ПКМ на Value и выборе Insert Keyframe (или при наведении на него курсора и нажатии горячей клавиши I), затем выбирается другой кадр на временной шкале, в поле ввода устанавливается новое значение Value и добавляется новый анимационный ключ для этого значения.

#

Анимация материалов

В отличие от Blender, в Verge 3D можно анимировать только определённый параметр материала, подключив к нему ноду Value или RGB.

Значения цветов заданные этими нодами могут быть анимированы в Blender нажатием ПКМ на соответсвующую ноду (или при наведении на него курсора и нажатии горячей клавиши I), затем выбирается другой кадр на временной шкале, устанавливается новое значение или цвет, которым добавляется анимационный ключ.

#

Процедурная анимация

Вместо того, чтобы проигрывать анимацию созданную в Blender, вы можете использовать Puzzles или JavaScript чтобы изменить координаты или другие параметры. В особенности animate param puzzle (основанный на Tween.js library) является удобным способом создания собственной анимации на лету.

#

Параметры воспроизведения

#

Настройки Verge3D для каждого объекта

Каждому объекту в Blender назначаются настройки Verge 3D, включая настройки воспроизведения анимации.

Настройки анимации:

  • Auto Start - отвечает за начало проигрывания анимации при загрузке сцены.
  • Loop Mode - режимы воспроизведения: Repeat - проигрывает анимацию от начала до конца, затем повторяет, Once - проигрывает один раз, затем останавливается, Ping Pong - проигрывает от начала до конца, затем воспроизводит анимацию в обратную сторону.
  • Repeat Infinitely - если необходимо, чтобы анимация объекта проигрывалась бесконечно. Действительно только для режимов Repeat и Ping Pong.
  • Repeat Count - отвечает за количество раз, которое будет проиграна анимация. Действительно только для Repeat и Ping Pong, если режим Repeat Infinitely отключен.
  • Offset - какое количество кадров должно быть пропущено перед началом воспроизведения анимации.

#

Интерполяция

Меню переключения интерполяции между ключевыми кадрами вызывается нажатием ПКМ в окне временной шкалы в Blender (горячая клавиша T).

Verge 3D поддерживет все режимы интерполяции доступные в Blender.

#

Частота смены кадров

Значение частоты смены кадров (по умолчанию - 24 fps) влияет на скорость воспроизведения и может быть изменено во вкладке Output Properties.

#

Контроль анимации с помощью Puzzles

Вы можете использовать animation puzzles для воспроизведения предварительно созданной анимации (для получения более подробной информации пройдите по ссылке).

Анимация также может быть реализована с использованием других блоков Puzzles - смотрите Camera и Time puzzles.
